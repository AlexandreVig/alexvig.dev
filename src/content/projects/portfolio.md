# Portfolio Website

The site you're looking at. A Windows XP–themed portfolio built as a tiny
desktop shell, with a virtual file system and multi-instance windows.

## Stack

- **Astro** for the static shell
- **TypeScript** throughout
- **Plain CSS** with CSS variables for theming
- **Cloudflare Pages** for hosting

## Architecture highlights

- Virtual file system with declarative folder tree
- Shell launcher that resolves file types to default apps
- Dynamic window manager — windows are created and destroyed at runtime
- Code-split per app and per content file
