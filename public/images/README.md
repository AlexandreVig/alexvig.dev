# Portfolio Image Placeholders

This folder should contain your project and blog images.

## Recommended Structure

```
images/
├── blog/
│   ├── astro-portfolio.jpg
│   ├── tailwind-themes.jpg
│   └── motion-one.jpg
├── projects/
│   ├── taskflow.jpg
│   ├── codesnippet.jpg
│   └── weathervue.jpg
└── profile.jpg
```

## Recommended Sizes

- **Hero images**: 1200x630px (optimized for social sharing)
- **Project thumbnails**: 800x450px (16:9 ratio)
- **Profile photo**: 400x400px (square)

## Tips

1. Use WebP format for better compression
2. Add meaningful alt text in your components
3. Consider using Astro's Image component for automatic optimization
